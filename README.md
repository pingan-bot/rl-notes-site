# 强化学习入门笔记网站

这是一个用于记录强化学习入门笔记的静态网站。

## 功能

- 新建、编辑、删除笔记
- 按模块和掌握度筛选
- 搜索标题、标签和正文
- Markdown 编辑与实时预览
- 深色 / 浅色模式切换
- JSON 导出 / 导入备份

## 在线部署

推荐使用 GitHub Pages：

1. 进入仓库 Settings
2. 点击左侧 Pages
3. Source 选择 Deploy from a branch
4. Branch 选择 main
5. 文件夹选择 /root
6. 点击 Save

部署完成后，访问：

```text
https://pingan-bot.github.io/rl-notes-site/
```

## 注意

笔记内容默认保存在当前浏览器本地 localStorage 中，不会自动同步到 GitHub 仓库。建议定期点击网页右上角的“导出备份”。
